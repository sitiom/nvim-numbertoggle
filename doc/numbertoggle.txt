1. nvim-numbertoggle                          *numbertoggle-nvim-numbertoggle*

Neovim plugin to automatically toggle between hybrid and absolute line numbers.
Written in Lua.

GETTING STARTED                                 *numbertoggle-getting-started*

REQUIREMENTS ~


- Neovim 0.7 or later


INSTALLATION ~

Use your favorite package manager. Example config with packer.nvim
<https://github.com/wbthomason/packer.nvim>:

>
    use {
       "sitiom/nvim-numbertoggle",
       config = function()
          require("numbertoggle").setup()
       end
    }
